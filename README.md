# imdb-2025-runtime-ratings-analysis

## Overview
This project explores IMDb movie data from 2025 to analyze the distribution of movie runtimes and ratings,
and to examine whether longer movies tend to receive higher ratings.

## Tools Used
- Python
- NumPy and Pandas
- Matplotlib and Seaborn

## Key Analysis
- Calculated average IMDb rating and runtime
- Visualized rating and runtime distributions using histograms
- Analyzed the relationship between runtime and rating using scatter plots
